ML Threat Classification

Random Forest model trained on 40+ attack patterns including DDoS, port scans, brute force, and exfiltration. 99.2% accuracy with real-time confidence scoring.

Active Prevention (IPS)

Quick Start
# Clone the repository
$ git clone https://github.com/Ansh5008/Smart_IntrusionDetectionSystem.git
$ cd Smart_IntrusionDetectionSystem

# Install dependencies
$ pip install -r requirements.txt

# Run as Administrator for IPS capabilities
$ streamlit run main.py

🛡️ CyberShield IDS/IPS is running at http://localhost:8501
